Capabilities

What OpenNavicat brings

⌨

CLI-First

40+ commands covering connection, query, schema, data, backup, and AI — all from your terminal.

🤖

AI-Native

Natural language to SQL, query optimization, schema design, and a ReAct agent that explores and executes.

📦

Dual Interface

Full CLI and optional PySide6 GUI with acrylic glass themes, table designer, and command recording.

🐬

MySQL + PostgreSQL

Async drivers (aiomysql + asyncpg) with connection pooling, SSH tunnels, and schema sync.

🔁

Sync & Migration

Schema compare, data sync, and dialect translation (MySQL ↔ PostgreSQL via sqlglot).

⏰

Automation

APScheduler-based job scheduling: automated backups, query execution, and sync tasks.

🔒

Privacy First

AI data masking (PII redaction), dry-run mode, DDL/DML confirmation dialog — your data stays yours.

🌐

i18n Ready

Fully translatable UI (1289 keys per language, en_US/zh_CN built-in).

Get Started

Install in one command

🐍

pip

Install from PyPI with optional extras.

pip install opennavicatpip install opennavicat[postgresql]
🐳

Docker

Multi-arch image with CLI and GUI.

docker pull ghcr.io/anomalyco/opennavicat:latest
📦

Standalone

Download CLI or GUI binary from releases.

curl -sSf https://opennavicat.dev/install.sh | sh
