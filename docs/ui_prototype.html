查询最近7天注册的用户数量，按日期分组
我 · 刚刚
我来帮你生成并执行这个查询。
SELECT DATE(created_at) AS 注册日期,
       COUNT(*) AS 注册用户数
FROM users
WHERE created_at >= DATE_SUB(CURDATE(), INTERVAL 7 DAY)
GROUP BY DATE(created_at)
ORDER BY 注册日期 DESC
✅ 查询完成，返回 7 条记录。
